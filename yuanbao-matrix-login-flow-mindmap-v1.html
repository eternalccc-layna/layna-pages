01 · Login Methods

登录方式定义

P0 上线 3 种 + 兜底 1 种；P1/P2 逐步开放。"默认推荐"决定登录页第一屏长什么样。

阶段 登录方式 默认推荐 触发体验 前置条件 / 失败回落
P0 一键登录 主推 识别 SIM 卡号 → 用户点"本机号码登录" → 0 秒完成 需 4G/5G + SIM 卡。失败 → 落短信
P0 微信授权 次推 跳微信确认页 → 返回 App 即登录 需安装微信。失败 → 落短信
P0 短信验证码 — 填手机号 → 收短信 → 输入 6 位 所有方式的兜底
P1 扫码登录 — 已登录端展示二维码 → 新端扫码确认 需主端在线。仅"端 → 端"，不替代首登
P1 QQ 授权 — 跳 QQ → 返回 App 即登录 需安装 QQ
P1 Apple ID iOS 必接 系统弹窗 → Face ID 确认 iOS 合规要求，App Store 必须提供
P2 厂商账号
华为/小米/OPPO/vivo		 — 系统级一键，体感同 Apple ID 仅厂商定制版默认开启
P2 邮箱 / Google — 邮箱 + 密码 / Google OAuth 出海版才开放

03 · State Machine

登录状态机：成功 / 失败 / 中间态

三类终态对应不同的 UI 反馈与重试策略，前端必须明确区分。

✓

成功态

  • 下发 Token + 写入本地
  • 主场景顶部 Toast「欢迎回来」
  • 无缝回到登录前操作
  • 头像 / 昵称右上角刷新
  • 资产同步进度条（可隐藏）
⋯

中间态（Loading / 等待用户）

  • 跳第三方授权页 → 灰底蒙层
  • 等待短信 → 60 秒倒计时按钮禁用
  • 风控人机验证 → 滑块 / 选图
  • 设备信任二次校验 → Face ID 弹窗
  • 超时 30 秒未返回 → 询问"是否重试"
✗

失败态

  • 网络异常 → 顶部 Toast + 重试按钮
  • 验证码错 → 输入框红描边 + 文案提示
  • 用户拒绝授权 → 静默回到 L2 面板
  • 风控拦截 → 弹窗 + 联系客服入口
  • 频率超限 → 倒计时锁 + 引导换方式
04 · Edge Scenes

8 个必须覆盖的细化场景

主流程之外的边角情况，每一个都对应一段独立的产品文案与交互。

SCENE 01 · 首登

新用户首次进入元宝

触发
App 启动 → 默认游客 → 触发付费/资产场景
体感
面板 → 一键登录 → 0.5 秒进主场景 → 头像默认 + 昵称"元宝用户"
关键
不强制设昵称、不强制绑微信，3 步内完成
SCENE 02 · 老用户回访

已登录用户重新打开 App

触发
App 启动 + Token 有效
体感
无任何登录页，直接进入主场景，资产后台同步
关键
90% 用户走这条路径，必须保证 0 秒入场
SCENE 03 · 跨端首登

已用 A 端登录，在 B 端打开

触发
B 端首次启动，触发登录
体感
面板顶部多一个"从手机扫码登录"，1 秒完成
关键
扫码后无需再填手机号、无需再选偏好
SCENE 04 · 多账号合并

同一手机号已绑过另一个微信

触发
用微信登录 → 服务端发现该手机号已存在另一 YuanbaoID
体感
弹窗"检测到已有账号，是否合并？"→ 二次确认
关键
合并不可逆，提供 30 天回滚通道
SCENE 05 · 一键登录失败

SIM 异常 / 飞行模式 / 双卡

触发
点一键登录 → 运营商网关返回失败
体感
Toast 提示 → 主按钮自动切换为"短信验证码"
关键
不让用户感知失败，无缝切兜底
SCENE 06 · 第三方拒绝

用户在微信确认页点取消

触发
跳微信 → 用户拒绝授权
体感
静默回到 L2 面板，不弹错误提示
关键
尊重用户选择，避免负反馈
SCENE 07 · 风控拦截

异地登录 / 短时多账号

触发
服务端识别风险 → 触发人机验证
体感
滑块 / 选图 → 通过后继续；不通过 → 24h 锁
关键
必须有客服入口，避免误伤投诉
SCENE 08 · 长期未用

30 天以上未登录

触发
refresh token 过期 + 设备信任过期
体感
App 启动 → 直接拉登录页（不报错）
关键
登录后资产无损恢复，给"欢迎回来"反馈

Key Decisions

登录流程的 3 个用户体感拐点

DECISION 01
登录页要不要默认勾选协议？
建议默认不勾。监管合规优先于转化率，被处罚的代价远高于增加的转化。
DECISION 02
新用户要不要强制设昵称？
建议不强制。默认"元宝用户 #1234"，让用户进主场景体验后自然产生设置动机。
DECISION 03
是否做"一键登录"主推？
建议主推。AppStore/工信部合规要求 + 比微信路径更短。失败时静默切短信。
